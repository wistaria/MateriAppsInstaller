# XCRYSDEN 

## SUMMARY 

 UNIX系プラットフォームで動作する結晶構造ならびにグリッドデータの可視化のためのオープンソースアプリケーション。GAUSSIAN, CRYSTAL, Quantum Espresso(PWscf), WIEN2k, FHI98MDなど、多くのソフトの出力ファイルに対応している。電子密度やポテンシャルなどの3次元データの可視化に加えて、フェルミ面の表示も行うことができる。
