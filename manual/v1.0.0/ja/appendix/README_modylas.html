MODYLAS¶

SUMMARY¶

任意の分子集合体に対する汎用の大規模分子動力学計算プログラム。長距離力の取り扱いを含め、ナノ分野・バイオ分野における分子動力学計算に必要な各種手法を備える。高効率の並列計算が可能であり、超並列スパコンを用いて溶媒中のウイルス(約1000万原子)の全原子計算やリポソーム(数十万原子)の長時間計算が可能。
