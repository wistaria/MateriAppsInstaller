# DCore 

## SUMMARY 

 動的平均場理論による多体量子計算を行うためのツール。予め定義されたモデルに加え、第一原理計算の結果からwannier90やRESPACKによって有効強束縛モデルを構成し、解析することが可能。計算結果を後処理することで、状態密度や波数空間でのスペクトル関数などを表示できる。計算にはTRIQSやALPSCoreなどの外部ライブラリを利用。
